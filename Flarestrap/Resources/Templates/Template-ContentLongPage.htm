Usage notes!

When inserting an image, use the class="img-mozu" for 75% size and responsive. Sections with ids are used for first level TOC headings. Sub sections as h3 and h4 are for nested sections. IDs are in those h3/h4s. The TOC supports only 2 levels, section with h2, and h3 in the list. Landing-title is the H1 with a different class name. Do not change the containers, scripts, etc. This file is not meant for compiling into the online help. It is strictly for standalone usage.
